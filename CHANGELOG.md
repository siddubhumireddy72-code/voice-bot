# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial project setup
- Flask web application with dashboard
- OpenAI GPT integration for conversational AI
- Asterisk VOIP integration for incoming calls
- Speech-to-Text using OpenAI Whisper API
- Text-to-Speech using gTTS
- Call logging and transcript storage
- Web dashboard for monitoring calls
- RESTful API endpoints
- Docker support
- Database models for calls, transcripts, and responses
- Configuration management system

### Features
- Real-time call handling
- Intelligent voice responses using GPT
- Multi-language speech support
- Call analytics and statistics
- Complete call transcripts
- User-friendly dashboard
- Database persistence
- Error handling and logging
